# REM 3.0 (Rust Extraction Machine)

Инструмент для автоматизированного извлечения кода (Extract Function) в Rust с поддержкой семантического анализа и базовым циклом исправления лайфтаймов.

## Текущие возможности

- **Семантический анализ (Rust-Analyzer)**: Сбор свободных переменных, определение выходных переменных и деструктуризация кортежей. Использует `ra-ap-ide` и `ra-ap-hir`.
- **Control Flow**: Обнаружение операторов `return`, `break` и `continue` в выделенном блоке.
- **Цикл исправления (Lifetime Repair)**: Итеративный прогон `cargo check` с попыткой модификации сигнатуры функции при возникновении ошибок заимствования (E0106).
- **Поддержка Async/Const**: Автоматическое определение необходимости модификаторов `async` и `const` для извлекаемой функции.

## Структура проекта

- `rem-domain`: Логика генерации кода, анализатор потока управления и механизмы исправления.
- `rem-application`: Координация процесса извлечения.
- `rem-infrastructure`: Адаптеры к подсистемам (Rust-Analyzer, Cargo CLI).
- `rem-cli`: Интерфейс командной строки.

## Использование

Извлечение функции по диапазону байт:
```bash
rem extract --file <PATH> --start <BYTE> --end <BYTE> --name <NAME> --project-root <ROOT>
```

## Тестирование

Запуск интеграционных тестов:
```bash
cargo test -p rem-cli --test integration
```
